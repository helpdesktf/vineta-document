Install by WordPress Theme Manager

This is probably the simplest way for most users. Here are the steps you need to take:

  1. Step 1 - Login to your WordPress Dashboard.
  2. Step 2 - Go to Appearance > Themes. Then click Add New button at the top of the screen.
  3. Step 3 - Click on the Upload button at the top.
  4. Step 4 - Click Choose File and select the themename.zip file.
  5. Step 5 - Click Install and wait a moment.
    6. Note:
    After installing the Main theme, if you want to use the Child theme, please install it the same way as above.
  6. Step 6 - Once installed, click Activate to enable the theme.
  7. Step 7 - A notice will appear at the top of the dashboard. Click Begin installing plugins.

    The theme requires these plugins: Contact Form 7, Elementor, Vineta Addon, WooCommerce, One Click Demo Import, MC4WP: Mailchimp for WordPress, WCBoost - Products Compare, WCBoost - Variation Swatches, WCBoost - Wishlist.
    Recommended: WPML, FOX - Currency Switcher Professional for WooCommerce.

  8. Step 8 - Go to Dashboard > Appearance > Install Plugins. Check all required plugins and activate them.
  9. Go to Appearance > Import Demo Data
    Select demo import
  10. Click Continue & Import
    And Wait

Install through FTP upload

If you want to install via FTP, here are the steps:

  • Using an FTP client, login to your server.
  • Navigate to the /wp-content/themes/ directory.
  • Upload the Vineta theme folder into that directory.

Once installation is complete, your Vineta theme will be ready to use.

Theme Option

To start customizing the theme, in the Admin page please go to Dashboard > Appearance > Customize.

Note:
Please remember to click Save & Publish after customizing to save changes.
These are global settings, so they affect the entire site. However, they can be overridden per post or page.

General

  1. Site Information

    To customize the theme, navigate to: Dashboard → Appearance → Customize → Site Identity

    • Site Title: Set or update your website’s title.
    • Tagline: A short description that summarizes your site.
    • Site Icon: The icon that appears in browser tabs, bookmarks, and WordPress mobile apps.
  2. Site Display

    To customize the theme, navigate to: Dashboard → Appearance → Customize → General

    • Enable Smooth Scroll: Toggle smooth scrolling on or off.
    • Enable Preload: Show a preloader while navigating between pages.
    • Go To Top Button: Enable or disable the "Back to Top" button for user convenience.

Header

To set up the default Header Style, navigate to Dashboard → Appearance → Customize → Header.

Custom Logo

ECommerce Features

1. Sale Countdown Timer

  1. Step 1: From the admin dashboard, navigate to Products → All Products.

  2. Step 2: Select the product you want to apply the sale countdown timer to.

  3. Step 3: In the Product data section, select Simple product as the product type.

  4. Step 4: In the General tab, click Schedule next to the Sale price field, then set the sale price and its valid date range.

For Products with Variations:

  1. Step 3: In the Product data section, select Variable product as the product type.

  2. Step 4: In the General tab, click Schedule next to the Sale price field, then set the sale price and its valid date range.

  3. Step 5: Configure the sale price, valid time range, and any other necessary details for each variation.

Step 6: Click Save changes after finishing your edits.

2. Products Frequently Bought Together

  1. To enable this feature, go to Customize → Single Product → Buy Together

    In this section, you can choose the display style (grid or list) and position (left or right).

  2. Select the product, then go to the Linked Product tab in the Product data section.

  3. Search for and add products to this section.

  4. Click Update to save your changes.

3. Product Comparison

To enable this feature, install and activate the WCBoost – Compare plugin.

After activation, a new settings section called Products Compare will appear under WooCommerce → Settings → Products → General. Use this section to configure how the comparison feature is displayed.

Click Update to save your settings.

4. Product Wishlist

To enable this feature, install and activate the WCBoost – Wishlist plugin.

Once activated, a new Wishlist tab will appear under WooCommerce → Settings. This tab lets you configure how the wishlist is displayed.

Click Save Options when you're done.

5. Product Currency

To enable this feature, install and activate the FOX - Currency Switcher Professional for WooCommerce plugin.

To change your shop's currency settings, go to WooCommerce → Settings → General → Currency Options, then select your desired currency from the dropdown menu.

Click Save Options after completing the setup.

6. Product Variation Swatches

Note: If you want to display the correct Attribute Type, you must set the Type first.

Create Variations

  1. Step 1: From the admin dashboard, navigate to Products → Attributes.

  2. Step 2: Fill in the attribute name (such as Size, Color, Image, etc.).

  3. Step 3: Determine how the attribute values are displayed using Type.

    You can display your attribute in 4 types:

    • Select Option
    • Colors
    • Label
    • Images

  4. Step 4: Click Add attribute when you're done. Then click on Configure terms to add new terms. The field will appear.

    You can now use this attribute when creating new products or apply it to existing ones.

Set Up Variation Swatches for Products

  1. Step 1: From the admin dashboard, navigate to Products → All Products.

  2. Step 2: Select the product you want to configure. In the Product data section, choose Variable product as the type.

  3. Step 3: Go to the Attributes tab and add the attribute you created earlier.

  4. Step 4: Enter attribute values, click Add new, and check both Visible on the product page and Used for variations. Then click Save Attributes.

  5. Step 5: Go to the Variations tab, click Generate variations from all attributes, and confirm.

  6. Step 6: All variations will now be generated. Select a default variation if desired and choose which attributes to show in the catalog.

  7. Step 7: Click each variation to set its image, price, sale countdown, and other relevant details.

  8. Step 8: Click Save changes and then click Update to save the product.

  3. Step 3: Go to Products → Variations and create image attributes.

  4. Step 4: In the Product → Variations tab, upload images for each variation.

9. Progress Sale

Creates a sense of scarcity, encouraging purchases (urgency/FOMO).

To enable this feature, go to Customize → Single Product → Progress Sale

The progress bar will only appear if you’ve set a product quantity and the item has had some purchases.

10. Size Guide

Help your customers shop with confidence. A clear size chart takes the guesswork out of buying a slim-fit dress—no try-on needed, no surprises.

To enable this feature, go to Customize → Single Product → People View

Add New Size Guide

Once you've enabled the size guide in your WooCommerce settings, a new menu item will appear: Products > Size Guides. On this page, simply click the “Add New” button at the top to create a new size guide for your products.

The key part of a size guide is the size table. In the Tables Box, use the plus/minus buttons to add or remove rows and columns. Click any cell to enter text directly.

In the product data box, a new Size Guide tab appears. Here, you can select a different size guide or adjust how it displays on the product page.

11. Volume Discount

Offer quantity-based discounts or set prices per quantity. Apply discounts at the cart, assign pricing tables to products or categories, and restrict by user roles.

To enable this feature, go to Customize → Single Product → Volume Discount

Add New Volume Discount

Once you've enabled the volume discount, a new menu item will appear: Products > Volume Discount. On this page, simply click the “Add New” button at the top to create a new volume discount for your products.

In this section, you can choose the layout (list or thumbnail), position, products, and set the Price Per Quantity Discount.

Click Publish to save your changes.

Pages

1. Create a Page

Step 1: Go to Pages in your admin sidebar and click Add New.

Step 2: Enter a page title, then in the Page Attributes box on the right, choose a Template.

2. Change Page Title

Step 1: Go to Pages → All Pages, hover over the page you want to update, and click Quick Edit.

Step 2: Update the page title and clear the current slug. Click Update to auto-generate a new slug.

You can also modify parent, template, status, etc. For more options, click Edit instead of Quick Edit.

3. Page Options

After creating a page, you can configure display options for it on the front end.

Example:

If the default header is Header Default, but you set Header Style 01 in Settings → TF Header → Header Style, and override it with Header Style 03 in the page options, then Header Style 03 will be used.

Step 1: Open the page in the editor and click Edit with Elementor.

Step 2: In the top center of the page, click the Settings icon.

Step 3: In Page Settings → TF Header, select your desired Header Style.

You can also adjust settings for the Logo, Topbar, Header, Page Title, Sidebars, and Footer.

Menus

How to Create a Menu

Step 1: Go to Appearance → Menu in the admin sidebar, then click Create a new menu.

Step 2: Enter a name for your menu and click Create Menu.

Step 3: Select the desired items from the left panel, click Add to Menu, then click Save Menu.

How to Create a Mega Menu

Step 1: Go to Appearance → Menu and click Create a new menu.

Step 2: Select existing menu items or add custom links.

Step 3: Click Enable Mega Menu.

If you haven’t created a Mega Menu template, go to Mega Menu and add a new template.

Customize your Mega Menu using elements from the sidebar, then click Publish.

Step 4: Once your Mega Menu is ready, return to Appearance → Menu. Click Enable Mega Menu, then set the width, position, and assign the Mega Menu template.

Step 5: Don’t forget to click Save Changes.
