Install by WordPress Theme Manager

This is probably the simplest way for most users. Here are the steps you need to take:

  1. Step 1 - Login to your WordPress Dashboard.
  2. Step 2 - Go to Appearance > Themes. Then click Add New button at the top of the screen.
  3. Step 3 - Click on the Upload button at the top.
  4. Step 4 - Click Choose File and select the themename.zip file.
  5. Step 5 - Click Install and wait a moment.
    6. Note:
    After installing the Main theme, if you want to use the Child theme, please install it the same way as above.
  6. Step 6 - Once installed, click Activate to enable the theme.
  7. Step 7 - A notice will appear at the top of the dashboard. Click Begin installing plugins.

    The theme requires these plugins: Contact Form 7, Elementor, Vineta Addon, WooCommerce, One Click Demo Import, MC4WP: Mailchimp for WordPress, WCBoost - Products Compare, WCBoost - Variation Swatches, WCBoost - Wishlist.
    Recommended: WPML, FOX - Currency Switcher Professional for WooCommerce.

  8. Step 8 - Go to Dashboard > Appearance > Install Plugins. Check all required plugins and activate them.
  9. Go to Appearance > Import Demo Data
    Select demo import
  10. Click Continue & Import
    And Wait

Install through FTP upload

If you want to install via FTP, here are the steps:

  • Navigate to the /wp-content/themes/ directory.

Theme Option

To start customizing the theme, in the Admin page please go to Dashboard > Appearance > Customize.

Note:
Please remember to click Save & Publish after customizing to save changes.
These are global settings, so they affect the entire site. However, they can be overridden per post or page.

General

  1. Site Information

    To Customize Theme, go to: Dashboard → Appearance → Customize → Site Identity

    • Site Title: You can set and change your site title.
    • Tagline: A short description of your site.
    • Site Icon: Icons that appear in browser tabs, bookmarks, and WordPress mobile apps.
  2. Site Display

    To Customize Theme, go to: Dashboard → Appearance → Customize → General

    • Enable Smooth Scroll: Enable or disable smooth scrolling.
    • Enable Preload: Show preloader when navigating between pages.
    • Go To Button: Enable or disable the “Go to Top” button.

Header

To set up the default Header Style, navigate to Dashboard → Appearance → Customize → Header.

  1. Custom Logo
    • Upload Your Logo: Drag or select your new logo here.
    • Upload Your Logo On Mobile: Drag or select your new mobile logo here.
    • Logo Size: Choose the size for the logo.
  2. Top Bar
    • Topbar: Enable or disable the top bar.
    • Social: Enable social icons, configured under the General tab.
    • Marquee: Enable scrolling text and input content to display.
    • Language: To use the language switcher, install and activate the WPML plugin (not included).
    • Currency: To enable the currency switcher, install and activate a third-party plugin. Vineta supports FOX – Currency Switcher Professional for WooCommerce.
  3. Navigation
  4. Header Options
    • Header Style: Choose the default header style.
    • Header Absolute: Enable or disable absolute positioning.
    • Header Sticky: Enable or disable sticky header.
    • Search Box: Enable or disable the search box in header.
    • Header Login/Register: Enable or disable login/register links.
    • Header Wishlist: Enable or disable wishlist icon.
    • Header Cart: Enable or disable shopping cart icon.
  5. Search Popup
    • Category: Choose the number of categories to display.
    • Product Type: Choose the number of product types to display.
    • Product Item: Choose the number of product items to display.
  6. Custom Menu

    To set up a custom menu layout, go to Dashboard → Appearance → Customize → Menus.

    • Select one menu to edit or create a new one.
    • Drag and drop items to rearrange the menu.

Page Title & Breadcrumb

    To set up a custom Page Title Style layout, navigate to Dashboard → Appearance → Customize → Page Title → Page Title.

  • Step 1: From the admin dashboard, navigate to WooCommerce → Settings → Advanced → Cart Page.

  • Step 2: Enable Quick View. You can also customize the appearance under Style Options.

  • Step 3: Click Save Changes after finishing your setup.

Shipping

  • Step 1: From the admin dashboard, navigate to WooCommerce → Settings → Advanced → Shipping.

  • Step 2: Set up Shipping Zones.

  • Step 3: Create a Free Shipping method and define the minimum order amount.

Coupons

To enable coupons in the Cart Drawer, Cart Page, and Checkout Page, go to WooCommerce → Settings → General and check the Enable coupons option.

2. Order

  • From the admin dashboard, navigate to WooCommerce → Orders.

  • You can view total orders on a daily, weekly, or monthly basis.

3. Coupons

  • From the admin dashboard, navigate to Marketing → Coupons.

  • Click the Add coupon button to create a new coupon.

  • Fill in the required information and click Publish to save the coupon.

ECommerce Features

1. Sale Countdown Timer

  1. Step 1: From the admin dashboard, navigate to Products → All Products.

  2. Step 2: Select the product you want to apply the sale countdown timer to.

  3. Step 3: In the Product data section, select Simple product as the product type.

  4. Step 4: In the General tab, click Schedule next to the Sale price field, then set the sale price and its valid date range.

For Products with Variations:

  1. Step 3: In the Product data section, select Variable product as the product type.

  2. Step 4: In the General tab, click Schedule next to the Sale price field, then set the sale price and its valid date range.

  3. Step 5: Configure the sale price, valid time range, and any other necessary details for each variation.

Step 6: Click Save changes after finishing your edits.

2. Products Frequently Bought Together

  1. Step 1: From the admin dashboard, navigate to Products → All Products.

  2. Step 2: Select the product, then go to the Bought Together tab in the Product data section.

  3. Step 3: Search for and add products to this section.

  4. Step 4: Click Update to save your changes.

3. Product Comparison

To enable this feature, install and activate the WCBoost – Compare plugin.

After activation, a new settings section called Products Compare will appear under WooCommerce → Settings → Products → General. Use this section to configure how the comparison feature is displayed.

Click Update to save your settings.

4. Product Wishlist

To enable this feature, install and activate the WCBoost – Wishlist plugin.

Once activated, a new Wishlist tab will appear under WooCommerce → Settings. This tab lets you configure how the wishlist is displayed.

Click Save Options when you're done.

5. Product Currency

To enable this feature, install and activate the FOX - Currency Switcher Professional for WooCommerce plugin.

To change your shop's currency settings, go to WooCommerce → Settings → General → Currency Options, then select your desired currency from the dropdown menu.

Click Save Options after completing the setup.

6. Product Variation Swatches

Note: If you want to display the correct Attribute Type, you must set the Type first.

Create Variations

  1. Step 1: From the admin dashboard, navigate to Products → Attributes.

  2. Step 2: Fill in the attribute name (such as Size, Color, Image, etc.).

  3. Step 3: Determine how the attribute values are displayed using Type.

    You can display your attribute in 4 types:

    • Select Option
    • Colors
    • Label
    • Images

  4. Step 4: Click Add attribute when you're done. Then click on Configure terms to add new terms. The field will appear.

    You can now use this attribute when creating new products or apply it to existing ones.

Set Up Variation Swatches for Products

  1. Step 1: From the admin dashboard, navigate to Products → All Products.

  2. Step 2: Select the product you want to configure. In the Product data section, choose Variable product as the type.

  3. Step 3: Go to the Attributes tab and add the attribute you created earlier.

  4. Step 4: Enter attribute values, click Add new, and check both Visible on the product page and Used for variations. Then click Save Attributes.

  5. Step 5: Go to the Variations tab, click Generate variations from all attributes, and confirm.

  6. Step 6: All variations will now be generated. Select a default variation if desired and choose which attributes to show in the catalog.

  7. Step 7: Click each variation to set its image, price, sale countdown, and other relevant details.

  8. Step 8: Click Save changes and then click Update to save the product.

Pages

1. Create a page

  1. Step 1: Navigate to Pages in your admin sidebar and click the Add New option.

    Step 2: Enter a new name for your page, then find the Page Attributes box on the right side to set your Template.

2. Change page title

  1. Step 1: Go to Pages > All Pages. Then hover over the page you want to edit. You will see the Quick Edit option.

    Step 2: Edit the title as you want. Delete the current slug, then hit Update. It will automatically generate a new slug for your page.

    You can also edit other options such as parent, template, status... If you want to edit more options, click Edit instead of Quick Edit.

3. Page Options

  1. After creating a page, you can set page options that prioritize displaying this page on the front end.

    For example:

    If the default header is Header Default but you choose Header Style 01 in Settings > TF Header > Header Style, then Header Style 03 will display on your page instead of Header Style 01.

    Step 1: In the page editor, click Edit with Elementor.

    Step 2: At the center top bar of the page, click the Settings icon:

    Step 3: In Page Settings > TF Header, choose the Header Style you want.

    You can also edit options for Logo, Topbar, Header, Page Title, Sidebars, and Footer.

Menus

How to create a menu

  1. Step 1: Navigate to Appearance > Menu in your admin sidebar and click the Create a new menu option.

  2. Step 2: Give your menu a name, then click Create Menu.

  3. Step 3: Choose menu items you want to add from the left and click Add to Menu. Don't forget to click Save Menu.

How to create a Mega Menu

  1. Step 1: Navigate to Appearance > Menu in your admin sidebar and click the Create a new menu option.

  2. Step 2: Select menu items or create custom link items.

  3. Step 3: Click Enable Mega Menu.

    If you haven't created a Mega Menu template yet, please navigate to Mega Menu and add a new template.

    Here you can customize the Mega Menu as you want by choosing elements from the left sidebar, then click the Publish button.

  4. Step 4: After you’ve finished creating your Mega Menu, navigate to Appearance > Menu. Click on Enable Mega Menu, then choose the width, position, and menu.

  5. Step 5: Don't forget to click Save Changes.

Blog Post

To Customize Post you go to: Dashboard -> Appearance -> Customize -> Content

  1. Blog archive
    • Sidebar Position: Select sidebar position on the posts page (No Sidebar/Left/Right)
    • Blog Layout: Select layout style for List (List/Grid)
    • Post Grid Columns: Choose to display the number of columns of the article
    • List Sidebar: In the dropdown box, select the sidebar you want to display on posts page
    • Post Content Elements: Drag and drop elements to re-order.
    • Post Excepts Length: This option is only available when you turn on "Auto Post Excepts". It uses to limit the content length which is shown on posts page
    • Social Share: Enable or disable Social Share
    • Read More Text: You can modify the text of Read more button here
    • Meta Elements: Drag and drop elements to re-order.
  2. Blog Single
    • Post Navigator: Enabled Post Navigator or not
    • Related Posts: Enabled Related Posts or not
    • Related Posts Style: Choose Related Posts Style (Blog List or Blog Grid)
    • Columns Of Related Posts: Choose 2 Columns, 3 Columns, 4 Columns
    • Number Of Related Posts: Number Of Related Posts
